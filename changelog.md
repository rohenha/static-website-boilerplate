# Changelog

## 1.0.0 - 2025-10-31

Release of the first version

### Features
- Eleventy integration
- Vite Js Integration
- JSX as template engine in Eleventy
- Tailwind installation
- Typescript configuration
- Creation of **vite-plugin-eleventy** to add eleventy files in ViteJs
- Creation of **html-perf** to optimize stylesheet and script tags in HTML
- Creation of **critical-css** to inline critical css in html files on build
- Set basic design
- Add blog structure
- Add sprite system for icons
- Add brotli plugin
- Add JS analyzer plugin
